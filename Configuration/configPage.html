External Ratings

Shows Letterboxd ratings for movies and IMDB ratings for TV shows directly in Jellyfin.

🎬 Letterboxd (Movies)

Letterboxd ratings are fetched by searching letterboxd.com. No API key required. Ratings are on a 5-star scale (e.g., 3.8 / 5).

📺 IMDB (TV Shows)

IMDB ratings are fetched via the free OMDB API. Get a free API key here (1,000 requests/day).

🎨 Display

⚡ Cache

Ratings are cached to avoid excessive requests. Clear the cache to force a refresh of all ratings.
